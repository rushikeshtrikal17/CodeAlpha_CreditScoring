# Credit Scoring Model using Machine Learning
(CodeAlpha Internship – Task 1)

## 📌 Objective
To predict an individual's creditworthiness using past financial data by applying machine learning classification techniques.

---

## 🧠 Approach
- Data preprocessing and feature engineering
- Classification using Logistic Regression
- Model evaluation using standard performance metrics

---

## 📊 Dataset
- Credit scoring dataset (CSV format)
- Features used:
  - Age
  - Sex
  - Job
  - Housing
  - Credit Amount
  - Duration
  - Saving Accounts
  - Checking Account
  - Purpose

---

## ⚙️ Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn

---

## 📈 Model Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1-Score
- ROC-AUC
